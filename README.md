Визуализация размерности инвариантных подпространств (tr(P))

Этот проект предназначен для анализа и визуализации изменения размерности инвариантных подпространств матрицы в зависимости от спектрального параметра.

Основной упор сделан на построение ступенчатого графика следа проектора (Trace of Projector).

Описание программРепозиторий включает в себя три ключевых компонента:

matre_x.m — Функция, которая задает исходную анализируемую матрицу.

S_S_D_M.m — Алгоритм на основе разложения Шура и метода Сильвестера для вычисления спектрального проектора $P$ и его характеристик.

plo.m — Скрипт визуализации, который строит график $tr(P)$ и отмечает точки скачков, соответствующие собственным значениям матрицы.

Математическая сутьГрафик функции $f(r) = tr(P_r)$ представляет собой ступенчатую функцию.

Каждый "скачок" на графике указывает на то, что сканирующий контур пересек одно или несколько собственных значений матрицы, что приводит к изменению размерности соответствующего инвариантного подпространства.
